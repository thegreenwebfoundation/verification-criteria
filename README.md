# Green Web Foundation Verification Criteria 

This repository contains copies of the Green Web Foundation's provider verification criteria in markdown format. This criteria is used to verify the green claims of hosting providers and data centers which seek to be [listed in the Green Web Dataset](https://www.thegreenwebfoundation.org/tools/green-web-dataset/get-verified/).

- Files starting with the `[current]` tag represent content which is part of the active verification criteria.
- Files starting with the `[draft]` tag represent new content which will either be added to/replace the current verification criteria.
- Files starting with the `[archived]` tag represent content which is no longer part of the verification criteria.
